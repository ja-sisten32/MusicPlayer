# MusicPlayer

Um player de música simples desenvolvido com HTML, CSS e JavaScript.  
O projeto demonstra a reprodução de áudio na web com interação do usuário.

## Preview

![Preview do projeto](preview.png)

## Sobre o projeto

Este projeto foi desenvolvido como exercício prático para demonstrar:

- Reprodução de músicas ao clicar nos cards
- Controle de Play e Pause
- Barra de progresso com tempo atual e duração
- Destaque visual da música selecionada
- Uso de eventos e manipulação do DOM em JavaScript

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Como executar

Baixe os arquivos do projeto e abra o arquivo `index.html`.  
Depois, clique em uma música para iniciar a reprodução.
